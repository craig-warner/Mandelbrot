Mandelbrot
========
- A multi-threaded C program for generating Mandelbort Images and Movies.

Building
========

Here are  the steps for building the executables for

 * cd Mandelbrot
 * cmake -S ./ -B apps/
 * cd apps
 * make -j

To run the program ./Mandelbrot
